# PLAN

This repository uses [`docs/PARITY_EXECUTION_PLAN.md`](docs/PARITY_EXECUTION_PLAN.md) as the
active implementation roadmap.

- It is the authoritative family-level execution plan.
- It records the phase, closeout artifact, and acceptance criteria for each parity
  family.
- Feature and behavior divergence must stay synchronized with `docs/FEATURE_PARITY.md`
and `docs/REFERENCE_PARITY_SURVEY.md`.
- Before claiming release readiness, required gates in `docs/PARITY_EXECUTION_PLAN.md`
  must pass for all relevant families.

Use this PLAN in priority over one-off notes when deciding what to implement next.
